## Description

c블라인드 배끼기

## Feature

- 인증 관리

  - 가입
  - 로그인

- 회원 관리

  - 탈퇴
  - 로그아웃
  - 프로필 조회
  - 프로필 사진 추가/수정
  - 프로필 사진 삭제

- 회사 관리
  - 회사 검색
  - 회사 등록
  - 회사 탈퇴
- 채팅
- 쪽지
- 게시물 관리

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm start

# watch mode
$ pnpm start:dev

# production mode
$ pnpm start:prod
```

## Test

```bash
# unit tests
$ pnpm test

# e2e tests
$ pnpm test:e2e

# test coverage
$ pnpm test:cov
```
